# LOne-structuringTheProjectDirectory

## Topic
File and folder structure for a React lesson project.

## What Changed From the Previous Lesson
This is the first lesson and establishes the base project structure for later lessons.

### New Files
- `src/main.jsx` — app entrypoint and root render logic.
- `src/App.jsx` — top-level app shell with core layout.
- `src/Components/Layout/Header.jsx` — shared header component.
- `src/Components/Elements/ProductCard.jsx` — reusable card component.
- `src/Pages/Home.jsx` — home page with product list.
- `src/data/products.js` — sample product data.
- `src/index.css` — global Tailwind theme and app styling.

### Modified Files
- None

### Removed Files (if any)
- None

## Key Concepts Introduced
- React project folder conventions
- Component and page separation
- Basic application shell
- Shared data module

## Why These Changes Were Made
This lesson sets up a stable project structure so learners can add features in later lessons without reorganizing the root layout.

## How to Run
```bash
npm install
npm start
```
